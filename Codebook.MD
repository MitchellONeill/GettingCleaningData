##The run_analysis.R file transforms the UCI HAR Dataset files into 2 datasets:
ии1. A dataset with the complete data on all subjects and activities relating to mean and std. MeanStdData (10299 obs, 81 variables)
ии2. A tidy dataset which displays only the average mean and std for each activity by each subject. TidyMeans (180 obs, 81 variables)

##Inorder to create dataset 1 the following actions were performed:
ии*The data from both the test and train dataset subfolders(for subject number, activity number and recorded output) are read into seperate data.frames: TrainData, TestData, TrainAct, TestAct, Train Suject, Test Subject
ии*The stored data from above are combined into 2 data.frames for FullTest and FullTrain
ии*The Activity and Feature files are read into data.frames: ActLables and ColNames
ии*The Feature(ColNames) data.frame is coerced to make the names more readable for R and then attached as column names to FullTest and FullTrain data.frames
ии*The two data.sets are combined by row into a data.frame "FullData"
ии*An index is created that lists the column numbers of the FullData data.frame that relate to recored mean and standard deviation values
ии*The index is used to subset "FullData" into a data.frame called "MeanStdData" that only inlcudes the indexed columns
ии*A for loop is used to replace all the numbered activity indicators with the names from the ActLables data.frame

##Inorder to create dataset 2 the following actions were performed:
ии*The MeanStdData data.frame is grouped first by Subject then Activity and finally summarised with the Mean function into a new data.frame: TidyMeans
